# 11_GLOSSARY.md - Terimler (MVP)

- **Pin (mafsal mesnet):** Düşey reaksiyon (R_A) ve moment taşımaz; yatay serbestlik kilitli.
- **Roller (makara mesnet):** Düşey reaksiyon (R_B); yatay yönde serbest, moment taşımaz.
- **Point Load (noktasal yük):** Kiriş üzerinde tek noktaya uygulanan kuvvet.
- **UDL (Uniform Distributed Load):** Belirli aralıkta sabit yoğunluklu yayılı yük.
- **Reaksiyon:** Mesnetlerin uyguladığı dengeleyici kuvvetler.
- **Kesme Kuvveti V(x):** Kiriş boyunca kesitlerde oluşan iç kuvvet; pozitif yukarı yön.
- **Moment M(x):** Kesitteki eğilme momenti; pozitif sagging (kiriş aşağı doğru bükülür).
- **Heaviside Fonksiyonu H(x):** Parça parça tanımlı yükleri matematiksel olarak ifade etmek için kullanılan basamak fonksiyonu.
- **KaTeX:** Tarayıcıda LaTeX denklemlerini hızlı render eden kütüphane.
- **Plotly:** Etkileşimli grafik çizimi için kullanılan JavaScript kütüphanesi.
